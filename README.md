# Local 825 Advanced Mode - Deployed Application

This is the production deployment of the Advanced Mode feature for Local 825's Bulldozer Dashboard.

## Features

- **Data Completeness Tracking**: Interactive matrix showing all companies and sections
- **Company Data Profile**: Detailed drill-down from company to data point level
- **Interactive Report Generation**: Company-by-company validation workflow
- **Intelligence Management**: Structured data storage with source attribution
- **Task Management**: Assignment and tracking of data collection tasks

## Access

This application is deployed and accessible via GitHub Pages.
